  [1X5.1 [33X[0;0YIntroduction[133X[101X
  
  [33X[0;0YQuivers  are  finite  directed  graphs.  Paths  in  a  given quiver [22XQ[122X can be
  concatenated  in  an  obvious  way,  and  this concatenation can be extended
  [22XK[122X-linearly (over a field [22XK[122X) to give an associative, unital algebra [22XKQ[122X called
  a  [13Xpath  algebra[113X.  A path algebra is infinite-dimensional iff its underlying
  quiver  [22XQ[122X  is  acyclic.  Finite-dimensional  [13Xquiver  algebras[113X  --  that  is,
  finite-dimensional  quotient  algebras  [22XKQ/I[122X  of  a  path algebra [22XKQ[122X by some
  (frequently  admissible)  ideal  [22XI[122X  --  are a very important class of rings,
  whose representation theory has been much studied.[133X
  
  [33X[0;0YThe  excellent  [5XQPA[105X  package  implements these objects in [5XGAP[105X. The (far more
  humdrum)  [5XSBStrips[105X  package extends [5XQPA[105X's functionality. Quivers constructed
  using  the [5XQPA[105X function [2XQuiver[102X ([14XQPA: Quiver no. of vertices, list of arrows[114X)
  belong to the filter [2XIsQuiver[102X ([14XQPA: IsQuiver[114X), and special biserial algebras
  are  those  quiver  algebras for which the property [2XIsSpecialBiserialAlgebra[102X
  ([14XQPA: IsSpecialBiserialAlgebra[114X) returns [9Xtrue[109X.[133X
  
  [33X[0;0YIn this section, we explain some added functionality for quivers and special
  biserial algebras.[133X
