  [1X2.1 [33X[0;0YHow to teach a special biserial algebra to [5XGAP[105X[101X[1X using quivers (from [5XQPA[105X[101X[1X)[133X[101X
  
  [33X[0;0YBefore discussing string modules, we have to specify a SB algebra to [5XGAP[105X. We
  use  functionality from the [5XQPA[105X package for this. Indeed, our [5XGAP[105X definition
  of  SB  algebra  is  an  object  for  which  [2XIsSpecialBiserialAlgebra[102X  ([14XQPA:
  IsSpecialBiserialAlgebra[114X) returns [9Xtrue[109X.[133X
  
  [33X[0;0YThe  first step is to define the presenting quiver [10Xquiv1[110X. We use just one of
  the  many methods [5XQPA[105X affords; see [2XQuiver[102X ([14XQPA: Quiver no. of vertices, list
  of arrows[114X) for details.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xquiv1 := Quiver( 2,[127X[104X
    [4X[25X>[125X [27X[ [ 1, 1, "a" ], [ 1, 2, "b" ], [ 2, 1, "c" ], [ 2, 2, "d" ] ][127X[104X
    [4X[25X>[125X [27X);[127X[104X
    [4X[28X<quiver with 2 vertices and 4 arrows> [128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe  second  step  is to create a path algebra [10Xpa1[110X. We need the quiver [10Xquiv1[110X
  that  we just created and some field. For convenience, take [10XRationals[110X (the [22Xâ„š[122X
  object in [5XGAP[105X).[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xpa1 := PathAlgebra( Rationals, quiv1 );[127X[104X
    [4X[28X<Rationals[<quiver with 2 vertices and 4 arrows>]>[128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe third step is to define a list [10Xrels[110X of relations over that path algebra.
  Relations are linear combinations of paths in [10Xpa1[110X. The addition, subtraction
  and multiplication of elements can be performed using [10X+[110X, [10X-[110X and [10X*[110X, as normal.
  Paths  are  [10X*[110X-products  of  arrows in [10Xpa1[110X. For details on refer to arrows of
  [10Xpa1[110X, see [2X.[102X ([14XQPA: . for a path algebra[114X).[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xrels := [ pa1.a * pa1.a, pa1.b * pa1.d,[127X[104X
    [4X[25X>[125X [27X             pa1.c * pa1.b, pa1.d * pa1.c,[127X[104X
    [4X[25X>[125X [27X             pa1.c * pa1.a * pa1.b, (pa1.d)^4,[127X[104X
    [4X[25X>[125X [27X             pa1.a * pa1.b * pa1.c - pa1.b * pa1.c * pa1.a ];[127X[104X
    [4X[28X[ (1)*a^2, (1)*b*d, (1)*c*b, (1)*d*c, (1)*c*a*b, (1)*d^4,[128X[104X
    [4X[28X  (1)*a*b*c+(-1)*b*c*a ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YWe  impose  a  rule  on  the  relations:  [13Xthey must be monomial relations or
  commutativity  relations[113X.  This  imposition  causes  no mathematical loss of
  generality, of course.[133X
  
  [33X[0;0YThe  fourth step is to create the ideal [10Xideal[110X generated by the relations and
  to tell [5XGAP[105X that the relations form a GrÃ¶bner basis of [10Xideal[110X.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xgb := GBNPGroebnerBasis( rels, pa1 );[127X[104X
    [4X[28X[ (1)*a^2, (1)*b*d, (1)*c*b, (1)*d*c, (-1)*a*b*c+(1)*b*c*a, (1)*c*a*b,[128X[104X
    [4X[28X  (1)*d^4 ][128X[104X
    [4X[25Xgap>[125X [27Xideal := Ideal( pa1, gb );[127X[104X
    [4X[28X<two-sided ideal in <Rationals[<quiver with 2 vertices and 4 arrows>]>,[128X[104X
    [4X[28X  (7 generators)>[128X[104X
    [4X[25Xgap>[125X [27XGroebnerBasis( ideal, gb );[127X[104X
    [4X[28X<complete two-sided Groebner basis containing 7 elements>[128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe  final step is to quotient [10Xpa1[110X by [10Xideal[110X, and hence finally obtain the SB
  algebra object [10Xalg1[110X[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xalg1 := pa1/ideal;[127X[104X
    [4X[28X<Rationals[<quiver with 2 vertices and 4 arrows>]/[128X[104X
    [4X[28X<two-sided ideal in <Rationals[<quiver with 2 vertices and 4 arrows>]>,[128X[104X
    [4X[28X  (7 generators)>>[128X[104X
  [4X[32X[104X
  
  [33X[0;0Y[13X(This algebra is the same as [13X[10XSBStripsExampleAlgebra( 1 )[110X[113X. See [14XA.2-1[114X for more
  information.)[113X[133X
  
  
  [1X2.2 [33X[0;0YHow to teach a string (graph) to [5XGAP[105X[101X[1X using strips.[133X[101X
  
  [33X[0;0YWe  continue  with  the  example  SB  algebra  [10Xalg1[110X, created in the previous
  section.[133X
  
  [33X[0;0YConsider  the following string (graph) for [10Xalg1[110X (given in non-LaTeX versions
  of this documentation as a formal word).[133X
  
  
  [24X[33X[0;6Y(a)^{-1} (b) (c) (a)^{-1} (b) (d)^{-1} (c) (a)^{-1} (c)^{-1} (d)[133X
  
  [124X
  
  [33X[0;0YReading  from  left to right, the first symbol in this string graph is [22Xa[122X and
  it  has exponent [22X-1[122X. It's followed by [22X2[122X arrows with positive exponent (which
  we  record  as  the  positive  integer  "[10X2[110X"),  then [22X1[122X with negative exponent
  (recorded  as  negative  integer  "[10X-1[110X"),  then  [22X1[122X positive ("[10X1[110X"), [22X1[122X negative
  ("[10X-1[110X"), [22X1[122X positive ("[10X1[110X"), [22X2[122X negative ("[10X-2[110X") and [22X1[122X positive ("[10X1[110X").[133X
  
  [33X[0;0YThis  is  the information used when specifying the string graph to [5XSBStrips[105X.
  The  operation  [2XStripify[102X  ([14X4.2-1[114X) returns the strip representing this string
  graph.  What  gets  printed  is  the  formal word associated to the original
  string graph.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xs := Stripify( alg1.a, -1, [ 2, -1, 1, -1, 1, -2, 1 ] );[127X[104X
    [4X[28X(a)^-1(b*c) (a)^-1(b) (d)^-1(c) (c*a)^-1(d)[128X[104X
    [4X[25Xgap>[125X [27XIsStripRep( s );[127X[104X
    [4X[28Xtrue;[128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe  reader  will  note that reading the string graph "from left to right" a
  moment  ago was only possible of how the graph was written on the page. That
  same  graph  may  be  written  equally  well  in  the "reflected" format, as
  follows.[133X
  
  
  [24X[33X[0;6Y(d)^{-1} (c) (a) (c)^{-1} (d) (b)^{-1} (a) (c)^{-1} (b)^{-1} (a)[133X
  
  [124X
  
  [33X[0;0YThis  gives  us different defining data for the string. However, [5XSBStrips[105X is
  smart enough to know that these two are representations of the same object.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xt := Stripify( alg1.d, -1, [ 2, -1, 1, -1, 1, -2, 1 ] );[127X[104X
    [4X[28X(d)^-1(c*a) (c)^-1(d) (b)^-1(a) (b*c)^-1(a)[128X[104X
    [4X[25Xgap>[125X [27Xs = t;[127X[104X
    [4X[28Xtrue[128X[104X
  [4X[32X[104X
  
  
  [1X2.3 [33X[0;0YHow to calculate syzygies of string( modules)s using strips[133X[101X
  
  [33X[0;0YWe  continue  with  the  strip  [10Xs[110X  for the SB algebra [10Xalg1[110X from the previous
  sections.[133X
  
  [33X[0;0YWe  know  that  [10Xs[110X  represents some (string) module [22XX[122X for [10Xalg1[110X. The syzygy of
  that  string module [22XX[122X is a direct sum of indecomposable string modules, each
  of  which may be represented by string graph. Those string graphs, or rather
  strips  representing  them,  can  be calculated directly from [10Xs[110X. This is the
  heart of the [5XSBStrips[105X package![133X
  
  [33X[0;0YSo,  let's  start  calculating  the "syzygy strips" of [10Xs[110X. This calls for the
  attribute  [2XSyzygyOfStrip[102X  ([14X4.5-1[114X),  which  returns a list of strips, one for
  each indecomposable direct summand of the syzygy of its input.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XSyzygyOfStrip( s );[127X[104X
    [4X[28X[ (v2)^-1(c) (a)^-1(b*c) (c*a)^-1(d^2), (a)^-1(v1), (d)^-1(v2) ][128X[104X
    [4X[25Xgap>[125X [27XLength( last );[127X[104X
    [4X[28X3[128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe  call to [2XLength[102X ([14XReference: Length[114X) reveals that the the syzygy of [10Xs[110X has
  [22X3[122X indecomposable summands.[133X
  
  [33X[0;0YOf  course,  there's  no reason to stop at [22X1[122Xst syzygies. [5XSBStrips[105X is able to
  take [22XN[122Xth syzygies very easily for [22XNâ‰¥0[122X. For example, we can calculate the [22X4[122Xth
  syzygy of [10Xs[110X as follows.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27X4th_syz := NthSyzygyOfStrip( s, 4 );[127X[104X
    [4X[28X[ (v2)^-1(c*a) (c)^-1(v2), (v2)^-1(d^2), (a)^-1(v1), (v2)^-1(d^2),[128X[104X
    [4X[28X  (a)^-1(b*c) (a)^-1(v1), (d^2)^-1(v2), (v1)^-1(a), (v2)^-1(v2),[128X[104X
    [4X[28X  (v2)^-1(c) (c*a)^-1(v2), (v1)^-1(a), (v2)^-1(v2),[128X[104X
    [4X[28X  (v2)^-1(c) (c*a)^-1(v2), (v2)^-1(v2), (a)^-1(v1),[128X[104X
    [4X[28X  (v2)^-1(c*a) (c)^-1(v2), (v2)^-1(d), (a)^-1(v1), (v2)^-1(d^2),[128X[104X
    [4X[28X  (a)^-1(v1), (d^2)^-1(v2) ][128X[104X
    [4X[25Xgap>[125X [27XLength( 4th_syz );[127X[104X
    [4X[28X20[128X[104X
  [4X[32X[104X
  
  [33X[0;0YWe find that the [22X4[122Xth syzygy of [10Xs[110X has [22X20[122X indecomposable direct summands.[133X
  
  [33X[0;0YThe  reader  will spot that many strips appear multiple times in [10X4th_syz[110X. If
  you  want  to  remove  duplicates,  then  the most efficient way is with [2XSet[102X
  ([14XReference:  Set[114X).  (Mathematically,  this  is  like  asking  for  just  the
  isomorphism  types of modules in the [22X4[122Xth syzygy of the module represented by
  [10Xs[110X, ignoring how often that type is witnessed.)[133X
  
  [33X[0;0YAlternatively, you can use [2XCollected[102X ([14XReference: Collected[114X), which turns the
  list  into  something that a mathematician might call a multiset. This means
  that  the  distinct  strips  are  recorded along with their frequency in the
  list.  For  example, the second output below means that [10X(v2)^-1(v2)[110X occurs [22X3[122X
  times in [10X4th_syz[110X while [10X(v1)^-1(a)[110X occurs [22X6[122X times.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XSet( 4th_syz );[127X[104X
    [4X[28X[ (v2)^-1(v2), (v1)^-1(a), (v2)^-1(d), (v2)^-1(d^2),[128X[104X
    [4X[28X  (v2)^-1(c*a) (c)^-1(v2), (a)^-1(b*c) (a)^-1(v1) ][128X[104X
    [4X[25Xgap>[125X [27XCollected( 4th_syz );[127X[104X
    [4X[28X[ [ (v2)^-1(v2), 3 ], [ (v1)^-1(a), 6 ], [ (v2)^-1(d), 1 ],[128X[104X
    [4X[28X  [ (v2)^-1(d^2), 5 ], [ (v2)^-1(c*a) (c)^-1(v2), 4 ],[128X[104X
    [4X[28X  [ (a)^-1(b*c) (a)^-1(v1), 1 ] ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YThis  package  uses  the  term  [13Xcollected  lists[113X for these multisets, and it
  offers  several  built-in functionalities for calculating collected lists of
  syzygies.  Principal  among  these  are  [2XCollectedSyzygyOfStrip[102X  ([14X4.5-3[114X) and
  [2XCollectedNthSyzygyOfStrip[102X ([14X4.5-4[114X).[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XCollectedSyzygyOfStrip( s );[127X[104X
    [4X[28X[ [ (a)^-1(v1), 1 ], [ (d)^-1(v2), 1 ],[128X[104X
    [4X[28X  [ (v2)^-1(c) (a)^-1(b*c) (c*a)^-1(d^2), 1 ] ][128X[104X
    [4X[25Xgap>[125X [27XCollectedNthSyzygyOfStrip( s, 4 );[127X[104X
    [4X[28X[ [ (v2)^-1(c) (c*a)^-1(v2), 4 ], [ (v2)^-1(v2), 3 ],[128X[104X
    [4X[28X  [ (v1)^-1(a), 6 ], [ (v2)^-1(d^2), 5 ], [ (v2)^-1(d), 1 ],[128X[104X
    [4X[28X  [ (a)^-1(b*c) (a)^-1(v1), 1 ] ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe author recommends that, if calculating [22Xk[122Xth syzygies for large [22Xk[122X (say [22Xk â‰¥
  10[122X), you use a Collected method. Details can be found in [14X6.1[114X.[133X
  
  
  [1X2.4 [33X[0;0YAside: How to calculate [22XN[122X[101X[1Xth syzygies efficiently for large [22XN[122X[101X[1X[133X[101X
  
  [33X[0;0YThis section is a short digression, more philosophical than computational.[133X
  
  [33X[0;0YA  central point of the author's doctoral studies (refer [All ]) is that the
  syzygies  of  a  string module should be arranged in a particular format. (A
  little  more  specifically,  they  should  be written into a certain kind of
  array.)  Most  of  the  time  this  format  does  not  print nicely onto the
  Euclidean  plane  so, sadly, there is little hope of [5XGAP[105X displaying syzygies
  in the most "optimal" way. The closest it can get -- which is not very close
  at  all,  frankly  --  is  the  list  format  returned  by  [10XSyzygyOfStrip[110X or
  [10XNthSyzygyOfStrip[110X.  However,  this format compresses lots into a single line.
  This loses information and becomes a very inefficient way to store data (let
  along   compute   with   them).   By   using   functions   like   [10XCollected[110X,
  [10XCollectedSyzygyOfStrip[110X  and  [10XCollectedNthSyzygyOfStrip[110X,  we lose what little
  information  the  list  presentation  holds  onto,  but  we  streamline  out
  calculations greatly.[133X
  
  [33X[0;0YTo  see  this,  let  [10Xs[110X be the example strip from above and consider the [22X20[122Xth
  syzygy  of  [10Xs[110X.  The  following calculation shows that it has [22X344732[122X distinct
  summands  (many  of  which will be isomorphic). On the author's device, this
  took over [22X2[122X minutes to perform.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XNthSyzygyOfStrip( s, 20 );;[127X[104X
    [4X[25Xgap>[125X [27Xtime;[127X[104X
    [4X[28X130250[128X[104X
    [4X[25Xgap>[125X [27XLength( last2 );[127X[104X
    [4X[28X344732[128X[104X

  [33X[0;0YCompare  this  with  a  [10XCollected[110X  approach,  wherein  the  [22X20[122Xth  syzygy was

  comparison,  and  as  a  small  boast, we also include times for the [22X2000[122Xth,
  [22X20000[122Xth and [22X200000[122Xth syzygies for comparison.[133X

  [1X2.5  [33X[0;0YHow  to  call  the strips representing simple, projective and injective[101X
  [1X(string) modules[133X[101X
  
  [33X[0;0YWe  continue  with  the  SB  algebra  [10Xalg1[110X  from before which, we remind the
  reader,  was  defined  in  terms  of the quiver [10Xquiv1[110X. There is nothing very
  special  about  the  running  example  strip  [10Xs[110X  that previous sections have
  focussed  on. The associated string module is certainly not canonical in any
  way.[133X
  
  [33X[0;0YHowever,  there  are  some string modules which really [13Xare[113X canonical for one
  reason or another, and which [5XSBStrips[105X has methods to call. This includes the
  simple  modules and, more generally, uniserial modules. It also includes the
  indecomposable  projective  or  injective  modules,  provided  that they are
  string modules (which is not guaranteeed).[133X
  
  [33X[0;0YTo  obtain  the  list  of  strips  that  describe  the  simple  modules, use
  [2XSimpleStripsOfSBAlg[102X ([14X4.3-1[114X).[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XSimpleStripsOfSBAlg( alg1 );[127X[104X
    [4X[28X[ (v1)^-1(v1), (v2)^-1(v2) ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe [10Xi[110Xth entry is the strip describing the [10Xi[110Xth simple module [22XS_i[122X.[133X
  
  [33X[0;0YThe  uniserial  modules are also string modules. They correspond to paths in

  [1X2.7 [33X[0;0YHow to turn a strip into a quiver representation[133X[101X
  
  [33X[0;0YA combinatorial approach is all very well and good but perhaps you really do
  want a module for your SB algebra. [5XSBStrips[105X can provide![133X
  
  [33X[0;0YFor  the  following,  recall  from  preceding  sections  the strip [10Xs[110X for the
  algebra  [10Xalg1[110X.  Further  recall that [10Xalg1[110X was defined in terms of the quiver
  [10Xquiv1[110X.[133X
  
  [33X[0;0YThe  strip  [10Xs[110X stands for a string module for [10Xalg1[110X. That string module can be
  modelled  as a representation of [10Xquiv1[110X. To obtain that quiver representation
  from [10Xs[110X, use [2XModuleOfStrip[102X ([14X4.5-5[114X).[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xmodule := ModuleOfStrip( s );[127X[104X
    [4X[28X<[ 6, 5 ]>[128X[104X
    [4X[25Xgap>[125X [27XPrint( module );[127X[104X
    [4X[28X<Module over <Rationals[<quiver with 2 vertices and 4 arrows>]/[128X[104X
    [4X[28X<two-sided ideal in <Rationals[<quiver with 2 vertices and 4 arrows>]>[128X[104X
    [4X[28X, (7 generators)>> with dimension vector [ 6, 5 ]>[128X[104X
  [4X[32X[104X
  
  [33X[0;0YYou  can  turn  a  list of strips into a list of modules using [2XModuleOfStrip[102X
  ([14X4.5-5[114X).[133X

    [4X[28X  <[ 0, 3 ]>, <[ 2, 0 ]>, <[ 0, 1 ]>, <[ 2, 2 ]>, <[ 2, 0 ]>,[128X[104X
    [4X[28X  <[ 0, 1 ]>, <[ 2, 2 ]>, <[ 0, 1 ]>, <[ 2, 0 ]>, <[ 2, 2 ]>,[128X[104X
    [4X[28X  <[ 0, 2 ]>, <[ 2, 0 ]>, <[ 0, 3 ]>, <[ 2, 0 ]>, <[ 0, 3 ]> ][128X[104X
  [4X[32X[104X
