  [1XAbstract[101X
  [33X[0;0YString  modules for special biserial (SB) algebras are represented by string
  graphs. Many modules related to a given string module, including its syzygy,
  transpose  and  vector-space  dual  and hence Auslander-Reiten translate and
  inverse  translate,  are  also  string modules. These related modules can be
  calculated  combinatorially  rather  than algebraically. [5XSBStrips[105X implements
  this  functionality  in  [5XGAP[105X,  representing  string graphs as objects called
  [13Xstrips[113X.  It  includes  some  tests  for associated properties such as syzygy
  type, delooping level and weak periodicity.[133X
  
  [33X[0;0Y[5XSBStrips[105X  also  includes  bookkeeping  functionality for multisets, which it
  calls  [13Xcollected  lists[113X,  and  it  integrates  with (and depends on) the [5XQPA[105X
  package for quiver algebras and their modules.[133X
